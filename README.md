# Тренажёр слов на React

Интерактивное приложение для изучения английских слов с последовательным показом и проверкой перевода.

## Описание приложения

Тренажёр слов - это React-приложение, предназначенное для эффективного запоминания английской лексики. Приложение последовательно показывает слова в случайном порядке, позволяя пользователю сначала попытаться вспомнить перевод, а затем проверить себя.

## Основные возможности

- Изучение 20 базовых английских слов
- Случайный порядок показа слов без повторений в рамках сессии
- Поэтапное отображение: сначала английское слово, затем перевод
- Отслеживание прогресса обучения
- Поздравительное сообщение после завершения всех слов
- Возможность начать тренировку заново

## Особенности реализации

Приложение разработано на React с использованием функциональных компонентов и хуков.

## Выводы

Разработан тренажёр для изучения английских слов, обеспечивающий эффективное запоминание лексики за счет последовательного показа и проверки знаний.
